Themius Extension v1

Welcome to Themius Extension created to automate and simplify the time it takes to apply my theme to echo.

This also allows for custom choices between what you want and what you don't want with this style.

I'm happy this has gotten so far as it is truely the best project I have started in a very long time that has actually gotten somewhere with a goal unlike most of my projects.
