Installing Extension

Stylus is a common Google extension created to edit website pages live without much hassle, it is a predecesor of Stylish without the premium requirement of stylish.

To install Stylus you must head to the Google Store and install it with the permissions required, if you do not trust this extension please do not continue this installmentation of it.

Click Add to Chrome or whatever browser you are using and allow it to install. Once it is done it will be ready to add code to any website your heart desires it to.
