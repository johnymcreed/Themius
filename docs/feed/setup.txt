<header type="45">Installing Extension</header>
<p>
    <b>Stylus</b> is a common Google extension created to edit website pages live without much hassle, it is a predecesor of <b>Stylish</b>
    without the premium requirement of stylish.
</p>
<p>
    To install Stylus you must head to the <a href="https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne">Google Store</a> and
    install it with the permissions required, if you do not trust this extension please do not continue this installmentation of it.
</p>
<p>
    <img class="allow-zoom" src="img/examples/extension.page.png" width="97.5%" height="700rem">
</p>
<p>
    Click <b>Add to Chrome</b> or whatever browser you are using and allow it to install. Once it is done it will be ready to add code to any website
    your heart desires it to.
</p>
<p>&ensp;</p>
