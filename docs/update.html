Themius self updater


This is the themius updater for your themius extension, it will automatically give you the newest version of themius and the instructions to install it.

Steps to install Themius

  • Extract to a Memorable place.
  • Go to your browsers extension page.
  • Enable developer mode (if needed).
  • Click Load unpacked.
  • Find the extension folder and add to browser.
